Проект представляет собой игру "змейка".

Есть 3 уровня:
1. Самый обычный, базовый уровень ( 10 очков для перехода )
2. Уровень со статическими препятствиями ( 15 очков для перехода )
3. Уровень с динамическими препятствиями ( появляются при поедании обычной еды ) ( 20 очков )

Основные игровые объекты: змейка, еда, препятствия:

Кроме обычной еды, есть золотая, которая дает бонусные очки за поедание,
но меняет свое местоположение каждые несколько секунд и пропадает на 
длительное время, если была съедена; золотая еда появляется не всегда, только при наборе
змейки определенной длины относительно последнего появления

Змейка увеличивается от поедания еды и ускоряется, исключением для ускорения 
является золотая еда ( не ускоряет ). Змейка умирает при выходе за границы экрана,
при самопоедании и при столкновении с препятствиями. Поедание обычной еды заставляет
ее появиться в любом доступном месте.

Препятствия представляют собой синие блоки.

В игре есть несколько жизней, третий, финальный уровень, прибавляет 3 жизни к счетчику,
так как является довольным сложным из-за возрастания скорости и появления новых препятствий

После построения змейки нужного размера, появляется промежуточный экран и переход на другой уровень

Управление на W A S D, так же упомянуто на стартовом экране
